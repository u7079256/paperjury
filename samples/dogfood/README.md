# Dogfood Sample

This folder contains a complete PaperJury dogfood run:

- `original_draft.pdf`: the manuscript before the run.
- `revised_draft.pdf`: the manuscript after the run.
- `RUN_REPORT.md`: English human-verified comparison report.
- `RUN_REPORT.zh-CN.md`: Chinese human-verified comparison report.

The source `.tex` is intentionally not included; this sample shows the before/after
artifact and the run report, not a build fixture.
